# UN AI Impact Assessment — Workflow Application Presentation

This is the presentation for the **UN AI Impact Assessment (AIIA) Workflow Application Prototype**, developed for the AI Impact Assessment Focus Group (June 2026).

## About

This 20-slide presentation covers:

- The problem with the current Excel/email-based AIIA process
- The digital workflow application prototype built to replace it
- **Role deep-dives**: Entity Focal Point, Contributor, Reviewer, Administrator, Governance Viewer
- Role activity map (swimlane diagram) showing who does what at each stage
- The 10 CEB AI Principles embedded in the assessment questionnaire
- End-to-end workflow from use case registration to reviewer decision
- RBAC capability matrix
- Dashboard and analytics panels
- Policy chat with RAG (retrieval-augmented generation)
- Technical architecture and Azure migration path
- Development roadmap and next steps

## Application

The full workflow application prototype is built with:

- **Next.js 15** (App Router, React 19, TypeScript)
- **Supabase** (PostgreSQL + Row-Level Security + Auth)
- **Tailwind CSS** + **shadcn/ui** with UN blue design tokens
- **pgvector** for RAG-powered policy chat
- **Azure migration-ready** adapter pattern
